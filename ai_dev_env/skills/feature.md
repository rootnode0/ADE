# SKILL: Feature Implementation
- **Project Alignment:** Always follow the existing project structure and folder naming.
- **Django Standards:** If adding a model, include `created_at` and `updated_at`. Complete the stack: Model -> Serializer -> Viewset -> URL.
- **Type Safety:** Use type hints for all new function signatures.
- **No Bloat:** Ensure logic is placed in services/utilities, not just views.
